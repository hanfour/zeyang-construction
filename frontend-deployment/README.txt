========================================
🚀 快速部署（3步驟）
========================================

步驟 1：設定環境變數
----------------------------------------
編輯 .env.production 檔案：
- 修改 VITE_API_URL 為您的 API 網址
- 例如：https://zeyanggroup.com.tw/api

步驟 2：上傳檔案到 FTP
----------------------------------------
使用 FileZilla 連線後：
1. 上傳 build/* 所有檔案到網站根目錄
2. 上傳 images/ 資料夾到網站根目錄
3. 上傳 .htaccess 到網站根目錄

步驟 3：測試網站
----------------------------------------
1. 開啟瀏覽器訪問您的網站
2. 清除快取（Ctrl+F5）
3. 確認網站正常顯示

========================================
⚠️ 重要提醒
========================================

1. API 網址設定
   必須正確設定 VITE_API_URL
   否則無法連線後端服務

2. 檔案上傳位置
   上傳 build 資料夾的「內容」
   不是上傳整個資料夾

3. .htaccess 設定
   這個檔案很重要！
   確保已上傳並正確設定

========================================
❓ 常見問題
========================================

Q: 看到網頁原始碼？
A: 檢查是否上傳了 build 資料夾內的檔案

Q: 網站空白？
A: 按 F12 查看錯誤，可能是 API 設定錯誤

Q: 圖片破圖？
A: 確認 images 資料夾已完整上傳

Q: 無法登入後台？
A: 檢查後端服務是否運行，API 網址是否正確
