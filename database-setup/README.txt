========================================
澤暘建設網站 - 資料庫安裝包
========================================

📦 包含檔案：
1. schema.sql          - 資料庫結構檔案
2. 初始管理員.sql      - 預設管理員帳號與範例資料
3. .env.example        - 環境變數設定範例
4. 資料庫安裝說明.md   - 詳細安裝說明文件

========================================
🚀 快速開始
========================================

1. 建立資料庫：
   CREATE DATABASE zeyang CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

2. 匯入 schema.sql 檔案（使用 phpMyAdmin 或命令列）

3. 匯入 初始管理員.sql 檔案

4. 設定環境變數（參考 .env.example）

========================================
📝 重要資訊
========================================

預設管理員帳號：
- 帳號：admin
- 密碼：admin123456
- 請登入後立即更改密碼！

資料庫需求：
- MySQL 8.0 或更高版本
- 字符集：UTF8MB4
- 排序規則：utf8mb4_unicode_ci

========================================
🔐 安全提醒
========================================

1. 請務必更改預設管理員密碼
2. 請務必更改 JWT 密鑰
3. 建議使用專用資料庫使用者（非 root）
4. 定期備份資料庫
